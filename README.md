# Markdown

## Reference a file within the repo

- ✅ `[words.txt](words.txt)` - [words.txt](words.txt)
- ❌ `[](words.txt)` - [](words.txt)
- ❌ `[words.txt]()` - [words.txt]()
- ❌ `(words.txt)` - (words.txt)
